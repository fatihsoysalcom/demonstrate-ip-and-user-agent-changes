# Demonstrate IP and User-Agent Changes

This example demonstrates how web servers identify clients based on their IP address and User-Agent header. It shows how to make requests with a default identity, with a custom User-Agent, and illustrates the code structure for routing requests through a proxy to change the apparent source IP. This helps understand the common tracking methods and why solutions like mobile proxies are considered for bypassing restrictions.

## Language

`python`

## How to Run

1. Ensure you have Python and the 'requests' library installed (`pip install requests`).
2. Run the script: `python main.py`
